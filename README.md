# conceptual-seo

Claude Code marketplace for WordPress and SEO tools.

## Plugins

### wordpress-crud

Full CRUD operations for WordPress pages via the REST API with session-based authentication.

See [wordpress-crud-plugin/README.md](./wordpress-crud-plugin/README.md) for details.

## Marketplace

This repository serves as a Claude Code marketplace. The `marketplace.json` at the root catalogs all available plugins.
